📋 二、模拟测试（部分精选题）

下面是精选的 15 道示例题，涵盖考试的主要知识点。
（完整模拟测试建议使用 Microsoft Learn 官方 Practice Test）

🧩 Part 1：云概念基础

Q1. 哪种云模型允许客户仅为其使用的计算资源付费？
A. CapEx 模型
B. OpEx 模型
C. 传统托管服务
✅ 正确答案：B

Q2. 下列哪项描述最符合 PaaS（Platform as a Service）？
A. 用户仅管理应用与数据层
B. 用户需管理操作系统与网络层
C. 用户只访问虚拟机控制台
✅ 答案：A

☁️ Part 2：核心 Azure 服务

Q3. 以下哪项 Azure 服务属于 IaaS？
A. Azure Virtual Machines
B. Azure SQL Database
C. Azure App Service
✅ 答案：A

Q4. Azure Storage 提供哪几种类型的存储？
A. Blob、File、Queue、Table
B. Document、Cache、Message、Object
✅ 答案：A

Q5. Azure 区域（Region）与可用性区域（Availability Zone）的关系是？
A. 每个 Region 包含多个 AZ
B. 每个 AZ 对应一个 Region
✅ 答案：A

🔐 Part 3：安全与治理

Q6. Azure RBAC 的主要作用是？
A. 分配权限给用户或组
B. 管理网络带宽
✅ 答案：A

Q7. Azure Policy 的作用是？
A. 控制资源创建的合规性
B. 定义虚拟机大小
C. 管理 DNS 区域
✅ 答案：A

Q8. 哪个服务可在资源层提供加密密钥管理？
A. Azure Key Vault
B. Azure Monitor
✅ 答案：A

Q9. 若需让本地数据中心安全访问 Azure VNet，推荐使用：
A. VPN Gateway 或 ExpressRoute
B. App Service Plan
✅ 答案：A

💰 Part 4：成本与治理

Q10. Azure 成本管理（Cost Management）的主要功能是：
A. 监控和分析云资源支出
B. 自动部署资源
✅ 答案：A

Q11. 以下哪项可帮助你计算 Azure 成本？
A. Azure Pricing Calculator
B. Azure Resource Graph
✅ 答案：A

Q12. 如果某服务 SLA 是 99.9%，月平均可接受的停机时间约为？
A. 43 分钟
B. 4 小时
✅ 答案：A

📊 Part 5：监控与治理实践

Q13. Azure Monitor 中 Metrics 与 Logs 的主要区别是？
A. Metrics 是时间序列数据，Logs 是事件记录
✅ 答案：A

Q14. 在虚拟网络中实现 Storage 私有访问，需配置：
A. Private Endpoint + Private DNS Zone
✅ 答案：A

Q15. Azure Security Center（Defender for Cloud）的作用是？
A. 威胁检测与安全建议
✅ 答案：A
